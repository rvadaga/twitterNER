README

Results will be stored in the provided directory as
1. predictions
2. scores
3. results.txt

results.txt file will have the following parameters
1. epoch
2. training loss
3. training accuracy
4. validation loss
5. validation accuracy
6. validation f1
7. test loss
8. test accuracy
9. test f1

To make any changes in parameters, use the constants.py file.

Requirements
1. gensim
2. tqdm
3. tensorflow-gpu
4. six
